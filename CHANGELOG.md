# 변경 이력 (Changelog)

이 프로젝트의 모든 주요 변경 사항은 이 문서에 기록됩니다.
형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)를 따릅니다.

## [1.1.0] - 2026-07-02

대규모 고도화 릴리스입니다. 캐릭터 리디자인부터 자동 업데이트, Claude Code 알림 연동까지 위젯의 핵심 경험을 새로 다졌습니다.

### 추가됨
- **가재(게) 캐릭터 리디자인**: 기존의 단순한 주황색 덩어리 대신, 집게발·더듬이·부채 모양 꼬리·걸어다니는 다리를 갖춘 가재/게 모티브 캐릭터로 전면 교체. 더듬이 사이에 금빛 Claude 스파클(✦) 추가.
- **테마 5종**: 클로드(주황) · 바다(파랑) · 숲(초록) · 포도(보라) · 자정(다크) 중 우클릭 메뉴에서 즉시 선택 가능.
- **말풍선 시스템**: 작업 시작/완료, 인사 등 다양한 순간에 말풍선으로 말을 거는 기능. 우클릭 메뉴에서 on/off 가능.
- **시스템 트레이 아이콘**: 더블클릭으로 위젯 보이기/숨기기, 우클릭으로 보이기/숨기기·업데이트 확인·종료 메뉴 제공. 작업 표시줄 버튼은 노출하지 않음.
- **오늘의 기록(사용 통계)**: 오늘/누적 코딩 시간과 세션 수를 집계해 말풍선으로 확인 가능. `%APPDATA%\ClaudeWidget\settings.json`에 영속 저장.
- **Windows 시작 시 자동 실행**: 관리자 권한 없이 사용자 단위 레지스트리 Run 키를 통한 자동 시작 지원.
- **설정 영속화**: 테마, 크기, 위치, 항상 위에 표시, 말풍선 on/off, 자동 업데이트 on/off 등 모든 사용자 설정이 재시작 후에도 유지됨.
- **GitHub Release 기반 자동 업데이트**: 실행 시 및 6시간마다 GitHub Releases API를 확인해 새 버전을 감지하고, 클릭 한 번으로 설치 파일을 내려받아 조용히(`/SILENT`) 설치·재실행. 자세한 내용은 [docs/AUTO_UPDATE.md](docs/AUTO_UPDATE.md) 참고.
- **Claude CLI 알림 연동**: 위젯이 Claude Code의 훅(hook) 핸들러 역할을 겸해, `Notification`/`Stop`/`SubagentStop` 이벤트를 말풍선 메시지로 표시. `--hook`, `--notify` 진입점 제공. 자세한 내용은 [docs/CLI_INTEGRATION.md](docs/CLI_INTEGRATION.md) 참고.
- **앱 아이콘**: 트레이·바로가기·설치 프로그램에 사용되는 전용 앱 아이콘 추가.

## [1.0.0] - 최초 배포

- Claude Code 실행을 감지해 함께 코딩하는 기본 개념의 데스크톱 펫 최초 버전.
- 단순한 주황색 블롭(blob) 형태의 캐릭터로, Claude Code CLI가 실행 중이면 타이핑 애니메이션을 보여주고 그렇지 않으면 대기 상태를 유지.
